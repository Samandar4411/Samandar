Build Your Perfect Busan Hike

Product Requirements Document (PRD)

View Live Project

By Samandar | Version 1.0 | May 22, 2025

1. Overview

Build Your Perfect Busan Hike is a mobile-friendly, interactive web app that recommends customized hiking trails in Busan based on user preferences like hike duration, difficulty, scenery, and season. It helps locals and tourists quickly find the ideal trail for any occasion.

2. Problem Statement

With so many trails in Busan, it can be hard to find one that fits your time, ability, and interests. This tool solves that by giving tailored trail suggestions with no research needed.

3. Target Users

  • Casual hikers and families
  • Tourists seeking scenic or cultural routes
  • Busan residents with limited time
  • Fitness and adventure lovers

4. Features

Core Features (MVP)

Feature Description
Interactive Questionnaire Multi-step quiz with time, difficulty, scenery, and season filters.
Custom Trail Recommendations Displays 3 trail matches based on user selections with match percentages.
Trail Metadata Shows trail difficulty, distance, scenery type, and best season.
Responsive Design Optimized for mobile and desktop devices.
Modern UI Uses emojis, gradients, and smooth transitions.

5. Stretch Goal (Extension)

Implemented: Share your recommendation (screenshot or copy).

Planned: Add an interactive map using Leaflet.js for trail visualization.

6. User Flow

  1. Landing Page – Welcome and start button
  2. Step 1 – Choose duration
  3. Step 2 – Choose difficulty level
  4. Step 3 – Choose what you want to see
  5. Step 4 – Choose season
  6. Results – Display 3 best trails with details

7. Technical Overview

Category Details
Frontend HTML, CSS, Vanilla JavaScript
Hosting GitHub Pages
Recommendation Logic Simple point-based matching algorithm
Planned API Leaflet.js for map integration

8. Goldilocks Check

Aspect Evaluation
Not Too Simple Includes interactive logic, matching engine, dynamic result page
Not Too Complex No backend needed, easy to maintain
Just Right Visually engaging, functional, and expandable

9. Future Improvements
